# AI Study Buddy - VibeState'25 Submission

## Problem Statement
Mobile App Track - Building intelligent, private AI experiences for students

## Our Solution
An AI-powered study companion that helps students learn more effectively through:
- Intelligent note-taking and organization
- On-demand AI tutoring for any subject
- Automatic quiz generation from study materials
- Pomodoro timer for focused study sessions
- Gamified learning with streak tracking

## Key Features
1. **AI Chat Tutor**: Ask questions and get instant, clear explanations
2. **Smart Note Taking**: Organize and review study materials
3. **Auto Quiz Generation**: AI creates practice questions from your notes
4. **Study Timer**: Pomodoro technique built-in for better focus
5. **Progress Tracking**: Streak counter to build consistent study habits

## Technology Stack
- React for UI
- Claude AI API for intelligent features
- Responsive mobile-first design
- Real-time AI integration

## Target Users
Students of all levels who want to study smarter, not harder

## Impact
- Reduces study stress through AI assistance
- Improves retention with automated quiz generation
- Builds consistent study habits with gamification
- Accessible anywhere via mobile browser
